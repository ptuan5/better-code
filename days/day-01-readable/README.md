# Day 1: Readable

## Theme

Write code you will understand six months later.

## Learning Focus

- organizing project directories
- naming variables clearly
- writing helpful comments and docstrings
- using linting and style guides

## Modules

### 1A: Project Structure

- lecture: why readable structure matters
- activity: reorganize a messy directory
- discussion: compare different structures

### 1B: Code-Level Readability

- lecture: naming, comments, docstrings, linting
- activity: rewrite confusing code
- discussion: explain tradeoffs in revisions

## Materials To Add

- messy project starter folder
- code snippet for rewrite exercise in both R and Python
