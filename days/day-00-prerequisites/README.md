# Day 0: Prerequisites

## Goal

Make sure participants can run either R or Python comfortably before the workshop begins.

## Topics

- R, Python, RStudio, VS Code, and Jupyter support
- Pre-workshop survey on language preference and typical workflows
- Environment checks and troubleshooting

## Materials To Add

- participant setup guide
- pre-survey form or questions
- troubleshooting checklist
- sample "hello world" files in R and Python

## Suggested Deliverables

- one setup page under `participant-setup/`
- one R smoke test under `r/`
- one Python smoke test under `python/`
