# Day 3: Reproducible

## Theme

Someone else, or future you, can rerun everything.

## Learning Focus

- documenting environments
- identifying hidden dependencies
- converting scripts into reusable components
- thinking in pipelines rather than manual steps

## Modules

### 3A: Environments

- concept: package versions and dependency drift
- demo: environment file and installation flow
- activity: reconstruct an environment from a partial dependency list

### 3B: Scripts to Pipelines

- concept: why manual execution order is fragile
- activity: define execution order and modularize one script
- optional stretch: sketch pipeline logic
