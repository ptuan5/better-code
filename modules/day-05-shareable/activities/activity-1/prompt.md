# Activity 1: Fix This Repo & Prepare It for GitHub

## Goal

Make a tiny analysis project understandable and reusable for a new collaborator.

## Files

- starter code: `r/starter.R` or `python/starter.py`
- example approach: `r/example.R` or `python/example.py`

### Task

Treat the starter script like something you are about to hand off to another researcher.

1. Read the file as if you have never seen it before.
2. Improve names, script headers, and output descriptions.
3. Make the entry point and output file obvious.
4. Draft 3-4 README-style lines that explain what the script does and how to run it.

### Questions to Answer

- what is this project for?
- what input data or values does it use?
- what output file does it produce?
- what command or step should a new user run first?
