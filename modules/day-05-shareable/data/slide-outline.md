# Slide Outline

## Slide 1: What Makes Code Shareable

- clear purpose
- easy entry point
- visible setup steps
- understandable outputs

## Slide 2: README Essentials

- project purpose
- required tools or dependencies
- how to run the work
- where to find important files

## Slide 3: GitHub Collaboration Basics

- commit history as project memory
- pull requests for review and discussion
- issues for questions and planned work

## Slide 4: Polishing for Reuse

- remove ambiguity
- improve naming
- document assumptions

## Slide 5: Activity Setup

- review communication gaps
- improve the materials
- prepare a short showcase
