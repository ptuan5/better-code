# Day 3: Reproducible

## Focus

Make it possible for someone else, or future you, to rerun the workflow.

## Folder Layout

- `data/`: discussion prompts, slide outline, and activity notes
- `r/`: R reproducibility exercise and cleaned version
- `python/`: Python reproducibility exercise and cleaned version

## Suggested Materials

- a short outline on environments and execution order
- one fragile workflow to clean up
- one cleaned reference solution in each language
