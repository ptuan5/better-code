# Notebook Template

Use this outline for either R Markdown, Quarto, or Jupyter notebooks.

## 1. Title and Purpose

- What question does this notebook answer?
- What dataset does it use?
- What output should it create?

## 2. Parameters

Put all user-editable values at the top.

- input path
- output path
- key analysis options
- random seed

## 3. Setup

- import packages
- set plotting theme or options
- validate inputs early

## 4. Load Data

- read files
- check dimensions and expected columns

## 5. Analysis

- keep sections short
- separate data cleaning, analysis, and visualization
- avoid unexplained magic numbers

## 6. Save Outputs

- save tables, figures, or processed data with explicit paths

## 7. Session Information

- package versions
- runtime notes
- date generated if relevant
