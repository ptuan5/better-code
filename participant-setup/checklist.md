# Setup Checklist

Participants should complete this before Day 1.

## Required

- Install either R or Python, depending on your main workflow.
- Install at least one editor: VS Code, RStudio, or Jupyter-compatible tooling.
- Confirm you can open a project folder and run a script or notebook.
- Confirm you can install packages in your chosen language.
- Confirm Git is installed and available from the terminal.
- Create or log into a GitHub account.

## Recommended Smoke Tests

- Run a short script that imports one common package and prints a message.
- Open a notebook and execute all cells from top to bottom.
- Clone this repository and confirm you can browse the files locally.

## What To Bring To The Workshop

- your laptop and charger
- access to GitHub
- one recent analysis script or notebook you would be comfortable improving during activities
