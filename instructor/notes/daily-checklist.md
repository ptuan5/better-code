# Daily Teaching Checklist

## Before Class

- confirm slides and starter files are ready
- test all activities from a clean state
- verify links, datasets, and package installs
- prepare backup screenshots or solutions

## During Class

- start with the "why" before the "how"
- keep examples short enough to explain in plain English
- leave time for discussion after each activity
- note where participants get stuck for later revisions

## After Class

- collect common questions
- update materials based on confusion points
- record timing adjustments for the next run
- capture strong participant examples for future teaching
